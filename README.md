# ModeloCajasCSSpinturaRothko-
Este proyecto crea una pintura al estilo Rothko usando HTML y CSS. Utiliza div para crear rectángulos con diferentes colores y tamaños, y aplica propiedades de CSS como border-radius, box-shadow, y transform para simular el efecto de pintura a mano. Cada rectángulo se rota y se blurea para lograr un aspecto artístico.
